# Генератор тестов

Учебный проект для создания и прохождения тестов.

**Автор:** Ирина Демиденкова (1 курс, «Фронтенд- и бэкенд-разработка»)

[Репозиторий](https://github.com/demi-irina/higher-web-practice-quiz-generator)

## Установка и запуск

### Установка зависимостей

```npm install```

### Запуск в режиме разработки

```npm run dev```

### Сборка для продакшена

```npm run build```

### Просмотр собранной версии

```npm run preview```
